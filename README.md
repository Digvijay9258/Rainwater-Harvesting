# Rainwater Harvesting System

A modern web application built with Next.js for rainwater harvesting management, analysis, and certification.

## Features

- 🌧️ **Rainfall Analysis**: Real-time rainfall checking and analysis
- 📊 **Analytics Dashboard**: Comprehensive data visualization and metrics
- 🏠 **Rooftop Calculator**: Calculate harvesting potential for your property
- 💧 **Water Quality Monitoring**: Track and analyze water quality metrics
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile devices
- 🔐 **User Authentication**: Secure login and user management
- 📜 **Jal Shakti Certification**: Generate and manage water harvesting certificates
- 💰 **Cost-Benefit Analysis**: Calculate ROI for harvesting implementation
- 🎮 **Interactive Features**: Including water conservation games and tips

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI
- **State Management**: React Hook Form
- **Data Visualization**: Recharts
- **Authentication**: Built-in auth system
- **Package Manager**: PNPM

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Digvijay9258/Rainwater-Harvesting.git
```

2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
